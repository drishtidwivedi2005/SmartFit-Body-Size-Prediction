# SmartFit Body Size Prediction

An AI-powered apparel size prediction system built using Machine Learning in R.

## Features
- Predicts clothing size using body measurements
- Uses Decision Tree Classification
- Brand-aware prediction system
- Supports multiple clothing categories
- Includes BMI-based feature engineering

## Technologies Used
- R
- RStudio
- caret
- rpart
- rpart.plot

## Dataset Features
- Height
- Weight
- Chest
- Waist
- Hip
- Age
- BMI
- Gender
- Brand
- Category
- Size

## Machine Learning Workflow
1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Train-Test Split
5. Decision Tree Model Training
6. Prediction
7. Evaluation using Confusion Matrix

## Sample Prediction

Input:
- Height: 175 cm
- Weight: 72 kg
- Brand: Zara
- Category: T-shirt

Predicted Size:
